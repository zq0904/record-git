# Vi | Vim [参考](https://www.runoob.com/linux/linux-vim.html)
## 命令模式
```
  // vim 启动 直接就进入命令模式 命令模式只提供一些基础的命令 因此仍要依靠底线模式输入更多命令
  vim filename // vim编辑文件 必须加filename 文件不存在最后保存时会直接创建这个文件
  // 切换命令
  i a o // 切换至输入模式 区别仅仅是光标位置 i当前光标 I光标所在行第一个非空格符 a光标下一个字符 A光标所在行最后一个字符 o光标所在行的下一行 O光标所在行的上一行
  : // 切换至底线命令模式
  // 光标移动
  x // 删除当前光标所在处的字符
  30→ // 光标移项右移动30个字符
  fn+← // 光标移项当前行的最前面
  1G // 移动到这个档案的第一行 gg效果等同
  G // 移动到这个档案的最后一行
  nG // 移动到这个档案的第n行
  control+f // 向下翻一页
  control+b // 向上翻一页
  // 查找和替换
  /我 // 光标之后查找 ?我 光标之前查找
  n // 重复前一个查找 N反向重复前一个查找
  :%s/我/你/gc // 底线命令模式 %也可换成1,$ 表从第一行到最后一行 将我替换成你 g表全局类似正则 c表替换前询问是否替换
  :1,10s/我/你/g // 底线命令模式 1-10行 将我替换成你
  // 删除 复制 粘贴 回退 反回退
  10X // 删除10个字符 X向前x向后 X与1X等价
  3dd // 从当前行向下删除3行 dd与1dd等价
  3yy // 从当前行向下复制3行 yy与1yy等价
  p // 将已复制的数据 p粘贴在光标下一行 P粘贴在光标上一行
  u // 回退
  control+r // 反回退
  . // 重复前一个动作 (如 yy复制了当前行)
```
## 输入模式
```
  esc // 退出输入模式 切换至命令模式
```
## 底线命令模式
```
  esc // 退出底线命令模式 切换至命令模式
  :w // 保存
  :q // 退出
  :q! // 强制退出 不保存
  :wq // 保存并退出
  :wq! // 保存 强制退出
  :ZZ // 没修改过直接退出 修改过保存退出
  :set nu // 显示行号
  :set nonu // 取消行号
```
