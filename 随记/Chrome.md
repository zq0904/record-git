## Chrome 查看覆盖率
  command + shift + p
  键入 Show Coverage
## Chrome 捕获快照
  command + shift + p
  选择 Capture full size screenshot 捕获 全屏 快照
  选择 Capture area screenshot 捕获 区域 快照
  选中node 选择 Capture node screenshot 捕获 node节点 快照
  选择 Capture screenshot 捕获 屏幕 快照
